DronorParty v0.22

DronorParty

A polyrhythmic drone instrument built around 8 Karplus-Strong strings. Each string is plucked on its own divisor of the global tempo, so the strings drift in and out of phase forever — a slow, evolving drone that never quite repeats.

How to play

  • Click START. The drone begins immediately at a default tuning.
  • Each string strip has its own controls: ratio (pitch), divisor (rhythm), attack, decay (ring-time), brightness, jawari, halo, detune, pan + gain.
  • Top bar: TEMPO sets the master clock. SYMP couples strings together — at low settings (3-10%) you get the tanpura "halo". EXCITE sets how hard each pluck drives the loop.
  • Right column: ring-mod, delay, reverb. Reverb is on by default.
  • Tuning system dropdown changes the chord. Tuning preset slots (1-4 factory, 5-6 user) save the full per-string state.

Keyboard

  • space — toggle audio engine on/off
  • 1–8 — manual pluck of string N

Notes

  • SYMP at high values is now amplitude-clamped — the v0.20 in-loop tanh limiter prevents the cross-coupling runaway that earlier versions could produce. If you ever do hear painful blow-up at any setting, flag it as a bug.
  • Per-string DECAY is the primary tail-shaper. The ATTACK slider only shapes how the noise burst ramps in — the string's natural ring takes over after that.

Recent changes

  • v0.21: SYMP re-enabled at moderate cap (0.05 at slider=100%) with the full safety stack: v0.19 send-side tanh limiter + v0.20.3 in-loop soft limiter. Stable at any slider setting.
  • v0.20.3: DECAY slider works again (loop limiter is identity below ±0.7 so loop gain = feedback exactly at normal levels), Jawari DC bias removed (zero-mean curve + DC-block highpass), reverb default wet 18%, IR uses true exponential decay, reverb TIME only rebuilds the IR on slider RELEASE so dragging doesn't cut the tail.
  • v0.20: 8 strings (was 5), FX column consolidated, in-loop SYMP safety limiter (later refined in v0.20.3), Jawari volume compensation, About panel.
  • v0.19: SYMP send-side clamp + cap drop, reverb default ON, larger non-module text.
  • v0.18.1: decay 0.993 default, ADSR widget → single ATTACK slider, halo audibility fix, SPEC.md committed.
  • v0.18: EXCITE master slider, Attack-only burst, Jawari + Halo turned up.
