BUFFER SHUFFLER — TEST CASE
A million turntablists. Captures a rolling 12-second buffer of your source and sprays grains back at any density / length / pitch / position / pan. Each parameter is a RANGE — every grain picks a fresh value within it. Drag a band laterally to shift, drag up to widen, drag down to collapse to a single value. Grab the gold edge handles to resize one side. Click any value to type.
