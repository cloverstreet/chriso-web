getDisplayMedia — pick a Tab in the Chrome dialog, check "Share tab audio".

YouTube / Spotify web: DRM/EME may deliver silent audio track.

Tab capture is a copy — original tab still plays to its normal output. To truly intercept audio (no pass-through), you need OS routing: BlackHole, Loopback, or TotalMix virtual bus.

Output starts muted (−60 dB). Raise output gain slowly — no feedback if capturing a different tab.
