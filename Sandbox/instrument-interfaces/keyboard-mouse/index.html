Instrument Interfaces — the computer keyboard as an instrument

Two orthogonal sides of the same puzzle. A layout decides how the physical keys are arranged into a pitch field; a note system decides what frequency each position sounds. Swap either independently. Pitch is exact-Hz Web Audio — no MIDI, no 12-note assumption. Type on your keyboard, or click the keys.

Why not piano- (or theremin-) on-a-screen?

Skeuomorphic touch instruments inherit the piano's 12-TET-specific, C-major-biased 7+5 topology while losing the physical key edges that let you find notes without looking. No haptics, finger occlusion of the target, no isomorphism. Useful for teaching the familiar; a dead end for new instruments. (Hunt & Wanderley: the mapping is the instrument — many-to-one mappings feel expressive, one-to-one feels sterile. NIME.)

Isomorphic layouts — same shape = same interval, everywhere

An interval becomes a fixed move on the grid, so a chord/scale shape transposes by sliding your hand — and survives retuning to any temperament. The layouts here:

  • Wicki–Hayden (concertina): right = whole tone, up = fifth. ref
  • Harmonic Table / Tonnetz (C-Thru AXiS): right = major third, up = fifth ⇒ up-left = minor third. ref
  • Jankó (1882 piano reform): whole tones along a rank, ranks offset by a semitone. ref

Hardware in this lineage: Lumatone (280 hex keys, Bosanquet generalized layout), LinnStrument, Striso, H-Pi Tonal Plexus (211 keys/octave), and the Array mbira.

QWERTY as instrument — prior art

The Piano / Tracker layout here is the FT2 → Renoise → FL Studio → Ableton convention: bottom rows = one chromatic octave (black keys on the offset keys above), QWERTY+number rows the octave above. Renoise · qwerty-hancock (JS lib) · OpenMPT: isomorphic layouts on QWERTY. Keys are read by event.code, so it works regardless of OS keyboard locale.

Note systems

Subsets of 12-TET (pentatonic, diatonic, whole-tone, chromatic); equal divisions worth playing (19-, 24-quarter-tone, 31-, 53-TET); just intonation (pure ratios); the non-octave Bohlen–Pierce (repeats at the 3:1 tritave); and measured gamelan pélog / sléndro cents tables. The clean architecture mirrors Scala .scl (the scale) × .kbm (which keys play it).

Each key is labelled by its nearest 12-TET note plus cents deviation (e.g. G4+50), with the exact frequency beneath — so even deeply microtonal pitches have a readable anchor. Colour = pitch class within the period.
