Instrument Interfaces

Experiments in mapping physical input geometry to pitch — for new instruments, not piano/theremin-on-a-screen skeuomorphs. Each link is a self-contained exploration. Two orthogonal sides throughout: the layout (key → position) and the note system (position → pitch).

  • Keyboard & Mouse
    The computer QWERTY keyboard as instrument. Selectable layouts (Piano/Tracker, Linear, Registers, Wicki–Hayden, Harmonic Table, Jankó) × note systems (chromatic, pentatonic, diatonic, just, 19/24/31/53-TET, Bohlen–Pierce, pélog, sléndro). Polyphonic ADSR synth.

Rough, always-changing, sometimes-broken — this is sandbox play. More explorations land here as we build them.
